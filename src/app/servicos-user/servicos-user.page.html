Serviços prestados Soroterapia:
É um tratamento de uma doença por meio de soro, ou seja, de anticorpos já preparados.
Intramuscular:
Na via intramuscular a vacina é introduzida dentro do tecido muscular.

Essa via é muito utilizada porque possui uma rápida absorção.

 Implantes Hormanais:
O implante hormonal é um método de tratamento para algumas doenças ginecológicas.
Mesoterapia:
Consiste na aplicação de medicamentos sob a pele.
